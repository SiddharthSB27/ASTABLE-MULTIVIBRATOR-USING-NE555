# Astable Multivibrator using NE555 Timer

## Overview
PCB design of an astable multivibrator circuit using the NE555 timer IC.
The schematic and PCB layout were designed in KiCad and verified
using DRC and 3D PCB visualization.

## Tools Used
- KiCad
